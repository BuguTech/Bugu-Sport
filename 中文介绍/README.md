## 项目介绍
   不咕约球是一款以校园体育为主题的微信小程序，用户主要集中于学生群体，包括高校学生和中小学学生等。</br>
   <img src="../images/main.PNG" width = "250"/>

## 开发目的
   * 为小众体育运动爱好者们提供一个寻找拥有相同兴趣的爱好者的平台。
   * 增加学校之间体育交流途径，通过体育赛事，让不同学校、校区的同学们拥有更多直接交流的机会，让同学们可以扩展自己的社交圈、运动圈。 
   * 尝试组织一年一度的校园间的体育比赛，打造成学校之间的名片。（参考牛津剑桥大学一年一度的赛艇比赛、哈佛耶鲁大学一年一度的橄榄球比赛）
   * 提高校园体育赛事的氛围和影响力，探索校园体育文化的形成与内涵。从而让同学们可以在校园内体验到更好的运动生活。
   * 增强校园体育专业性，尝试将专业资源引入校园，探索校园体育为职业体坛扩展选材面积的可能性。（参考美国校园篮球发展，日本校园足球发展。） 
   * 学习借鉴NCAA等成熟的学生体育联盟，探索校园体育的商业价值。


## 功能介绍
### 约赛功能——发送邀请、协商比赛、跨校交流
   1. 同学们可以选择自己感兴趣的运动类型、所在的学校、校区，筛选出和自己有相同体育爱好的同学朋友。
   2. 并可以以个人身份或者队伍身份向想要进行比赛的同学队伍发出约赛申请

## 未来期望
   * 希望可以和更多学校体育部门协会合作，打造一个氛围更好凝聚力更强的校园体育社区，为同学们提供一个体验更好的社区平台。
   * 希望引入专业的资源，如教练员、裁判员，为同学们提供专业指导，建立球探体系，挖掘探索更多具有潜力的球员。

## 开发者心得
   * 作为一名普通的大学生、程序猿，同时也是一名体育爱好者，对于校园体育的发展有自己的一点看法，非常希望通过自己所学所感，为校园体育的发展贡献自己的一份绵薄之力，提高校园体育的关注度，让校园体育氛围更浓厚，让同学们也有机会获得更多和不同校区、不同学校用有相同兴趣爱好者进行比赛交流的途径。
   * 开发这款软件，一来是为了探索校园体育的种种可能，二来是为了在实践中找到灵感，从而在未来更有机会和经验为校园体育的发展助力。
   * 由于这款软件也是我们第一次进行团队项目开发和尝试，团队规模很小，开发的这款应用仍有种种不足，仍在不断迭代中。
   * 经过一段时间对校园体育的探索，我们也发现了诸多问题以及阻力，但我们仍然希望能够助力校园体育的发展，也真心希望越来越多的人关注校园体育、投身到校园体育当中，为国内体育的发展提供更多可能。
